Testing Instructions

Use this page to test the wallet functionality. The iframe on the right shows the app running locally.

Test Flows:

  1. Create a new wallet - Test the full flow from seed phrase to wallet dashboard
  2. Verify each step works correctly - Make sure validation and navigation work as expected
  3. Test error cases - Try invalid usernames, etc.

Common Issues:

  • If the app doesn't load, make sure the development server is running with npm run dev
  • Check for console errors in the browser developer tools
  • If the navigation doesn't work, check the React Router configuration
